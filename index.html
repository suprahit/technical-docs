Inicio

Python é uma linguagem de programação orientada a objetos, interpretada, de tipagem dinâmica de alto nível com propósito geral. Foi lançada em 1991 por Guido van Rossum.

A linguagem foi projetada para priorizar a legibilidade do código sobre a velocidade ou expressividade. Combina uma sintaxe concisa e clara com os recursos poderosos de sua biblioteca padrão e por módulos e frameworks desenvolvidos por terceiros.

O nome Python teve a sua origem no grupo humorístico britânico Monty Python, criador do programa Monty Python's Flying Circus.

Filosofia

O Zen of Python é uma coleção de 19 princípios orientadores, na forma de poema, com uma série de aforismos, para escrever programas de computador que influenciam o design da linguagem de progração Python.

Os príncipios que constam no Zen of Python são:

  • Bonito é melhor que feio.
  • Explícito é melhor que implícito.
  • Simples é melhor que complexo.
  • Complexo é melhor que complicado.
  • Linear é melhor que aninhado.
  • Esparso é melhor que denso.
  • Legibilidade conta.
  • Casos especiais não são especiais o bastante para quebrar as regras.
  • Ainda que praticidade vença a pureza.
  • Erros nunca devem passar silenciosamente.
  • A menos que sejam explicitamente silenciados.
  • Diante da ambiguidade, recuse a tentação de adivinhar.
  • Deve haver um - e preferencialmente apenas um - modo óbvio para fazer algo.
  • Embora esse modo possa não ser óbvio a princípio a menos que você seja holandês.
  • Agora é melhor que nunca.
  • Apesar de que nunca normalmente é melhor do que *exatamente* agora.
  • Se a implementação é difícil de explicar, é uma má ideia.
  • Se a implementação é fácil de explicar, pode ser uma boa ideia.
  • Namespaces são uma grande ideia - vamos ter mais dessas!
Começando

Instalação de Python

Para conferir se você tem python instalado no PC com Windows, procure na barra de pesquisa por Python ou coloque o seguinte comando no Prompt de Comando (cmd.exe):

C:\Users\Your Name>python --version

Para conferir se você tem python instalado no Linux ou Mac, no linux abra o CLI ou no Mac abra o terminal e digite:

python --version

Se você não tiver Python instalado no computador, você pode baixá-lo de graça no site oficial do python.

Primeiro código

Python é uma linguagem de programação interpretada, isso significa que, como um desenvolvedor, você escreve arquivos Python (.py) em um editor de texto e então coloca esses arquivos em um interpretador de python para serem executados.

Assim se executa o código python na interface de linha de comando:

helloworld.py

print("Hello, World!")

Simples assim. Salve o seu arquivo. Abra o terminal, navegue até o diretório que você salvou o seu arquivo e rode:

C:\Users\Your Name>python helloworld.py

A saída deverá ser:

Hello, World!

Você acaba de executar o seu primeiro programa Python!

A linha de comando Python

Para testar uma pequena quantidade de código em Python, talvez seja melhor escrever o código diretamente na linha de comando.

Digite o seguinte no Terminal:

C:\Users\Your Name>python

Ou, se o comando "python" não funcionou, tente "py":

C:\Users\Your Name>py

A partir daí, você pode escrever qualquer código python, incluindo o exemplo de hello world acima.

Quando você terminar de testar seu código, digite "exit()" para sair da interface de linha de comando do python:

exit()
Sintaxe Python

Indentação Python

A indentação se refere à quantidade de espaços antes do começo de uma linha de código.

Em outras linguagens de programação, a indentação serve somente para legibilidade, mas no python, a indentação é muito importante, pois, o python usa a indentação para indicar um bloco de código.

if 5 > 2:
  print("Five is greater than two");

Python gerará um erro se você errar a indentação:

if 5 > 2:
print("Five is greater than two");

A quantidade de espaços fica por você, porém, não é recomendado utilizar mais de 4 espaços. O TAB também pode ser usado.

Variáveis

Variáveis são um tipo de contêiner na memória para guardar valores

Criando variáveis

Python não possui um comando para declarar uma variável.

Uma variável é criada no momento que você declara um valor a ela.

x = 5
print(x);

No Python, você também, não precisa declarar nenhum tipo para a variável, e pode até mudar o tipo da variável depois que ele foi definido.

x = 5
x = "Sally"
print(x);

Casting

Se você quiser especificar o tipo de uma variável, você usará o casting.

x = str(5)
x = int(5)
x = float(5)

Maiúsculas e Minúsculas

O nome das variáveis difere maiúsculo e minúsculo.

a = 5
A = "Sally"

"A" não irá sobrescrever "a".

Nome de Variáveis

Uma variável pode ter um nome curto (como x ou y) ou um nome mais descritivo (idade, volume_total). Essas são as regras para nomes de variáveis no python:

  • O nome de uma variável deve começar com uma letra ou _
  • O nome de uma variável não deve começar com um número
  • O nome de uma variável só pode conter caracteres alfa-numéricos e underscores (A-z, 0-9, e _)
  • Nomes de variáveis diferem maiúsculo e minúsculo(idade, Idade e IDADE são três variáveis diferentes)
  • O nome das variáveis não pode ser nenhuma das Palavras-Chave do Python.
